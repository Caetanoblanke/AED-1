Problema 143 LeetCode - Reorder List

É dado o cabeçalho de uma lista simplesmente encadeada. A lista pode ser representada como:

L 0 → L 1 → … → L n - 1 → L n
Reordene a lista para que fique no seguinte formato:

L 0 → L n → L 1 → L n - 1 → L 2 → L n - 2 → …
Você não pode modificar os valores nos nós da lista. Somente os próprios nós podem ser alterados.

Restrições:

O número de nós na lista está no intervalo .[1, 5 * 104]
1 <= Node.val <= 1000
