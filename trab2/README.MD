Leetcode 498 - Diagonal Transversa

Dada uma m x nmatriz mat, retorne um array contendo todos os elementos da matriz em ordem diagonal .

Exemplo 1:


Entrada: mat = [[1,2,3],[4,5,6],[7,8,9]]
 Saída: [1,2,4,7,5,3,6,8,9]
Exemplo 2:

Entrada: mat = [[1,2],[3,4]]
 Saída: [1,2,3,4]
 

Restrições:

m == mat.length
n == mat[i].length
1 <= m, n <= 104
1 <= m * n <= 104
-105 <= mat[i][j] <= 105
